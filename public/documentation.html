GET /movies/:title 200(success) 400(fail) Shows data from title
GET /movies/genres/:genreName 200(success) 400(fail) Shows data on a genre
GET /movies/directors/:directorName 200(success) 400(fail) Shows data on a director
PUT /users/:Username 200(success) 400(fail) Allows user to update profile
DELETE /users/:Username/:movieTitle 200(success) 400(fail) Allows user to remove movie from list
DELETE /users/:Username 200(success) 400(fail) Allows user to remove email from registry
